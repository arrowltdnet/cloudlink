


## Cloudlink

CloudLink SaaS
CloudLink SaaS is a versatile and flexible Software-as-a-Service (SaaS) platform designed to deliver efficient IT solutions for businesses and organizations.

<br/ >
With CloudLink SaaS, users can easily access and use software applications remotely over the internet, without the need to install or maintain any software on their personal comcloudlinks.

### For Users
Cloudlink's goal is to provide you with every app and feature you need to work, create, and play under one roof. Cloudlink wants to be the all-in-one solution for your digital life.


### For Developers

Cloudlink provides everything you need to build and publish web apps and games.

<br/>

## Getting Started


### 💻 Local Development

```bash
git clone https://github.com/arrowltdnet/cloudlink
cd cloudlink
npm install
npm start
```
**→** This should launch Cloudlink at  http://localhost:4100


<br/>

### 🚀 Self-Hosting

#### Linux/macOS

```bash
curl -fsSL https://cloudlink.arrowltdnet.com/selfhost | sh
```

#### Windows

```powershell
irm https://cloudlink.com/selfhost?os=windows | iex
```

**→** For more details, see [Self-Hosting Cloudlink](./doc/self-hosting.md).

<br/>
